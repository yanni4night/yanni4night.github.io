使用 Date 构造一个指定日期，即可以传入毫秒单位的 UNIX 时间戳，也可以传入一定格式的字符串。

该格式在 ECMAScript 中的定义位于 15.9.1.15，它与 HTML5 使用的 RFC3339类似，区别参考 stackoverflow。
